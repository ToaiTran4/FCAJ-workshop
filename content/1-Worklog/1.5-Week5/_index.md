### Week 5 Objectives:
* Learn how to grant permissions for applications to access AWS services through IAM Role.
* Practice using Access Key and IAM for applications to upload files to Amazon S3.
* Get familiar with AWS Cloud9 (Cloud IDE in browser).
* Practice Lab5: host static website with Amazon S3.
* Learn how to control access with IAM Policies and Bucket Policies (recommend disabling old ACL).
* Deploy Amazon CloudFront in front of S3 bucket to increase security and access speed with CDN.
