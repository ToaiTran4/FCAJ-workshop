### Mục tiêu tuần 9:
* Tìm hiểu AWS Systems Manager: quản trị hàng loạt không cần SSH từng máy (Fleet Manager), cập nhật OS (Patch Manager), chạy lệnh từ xa (Run Command).
* Thực hành Lab9: cấu hình Session Manager (tạo Role AmazonSSMManagedInstanceCore, tạo VPC Endpoints, kết nối qua UI).
* Tìm hiểu khởi tạo hạ tầng dưới dạng code với AWS CloudFormation.
* Tìm hiểu tổng quan các dịch vụ bổ sung: AWS WAF, Storage Gateway, Amazon Cognito, ECS/ECR, SQS, Textract, EventBridge, SES/SNS, CodeCommit/CodeBuild/CodeDeploy.
