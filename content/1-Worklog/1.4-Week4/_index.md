### Week 4 Objectives:
* Learn in-depth about EC2: AMI (Amazon Machine Image) and how it works (Snapshot + metadata, doesn't store configuration since it's not on the drive).
* Learn about Amazon EBS Volume and automating snapshots with AWS Lifecycle Manager.
* Practice Lab4: create instance from custom AMI.
* Practice recovering access when losing key: use SSM to reset password on EC2 Windows, use User Data on EC2 Linux.
* Learn how to limit resource usage through IAM Policy: limit by Region, by Instance Family, by Instance Type, limit EBS Volume type, delete resources by IP and by time.
